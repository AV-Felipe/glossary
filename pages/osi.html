Open Systems Interconection

Modelo criado pela ISO em 1977, com o propósito de fornecer orientações sobre o funcionamento de sistemas de rede, tanto do ponto de vista do hardware como do software envolvido nesse sistema. Tal modelo é dividido em 7 camadas (mais referidas pelo nome em inglês layers) estratificando as funções dos diferentes componentes do sistema.

O modelo OSI não tem a função de definir uma implementação de sistema de rede específico, mas é um modelo conceitual que se presta como um guia para o desenvolvimento de sistemas específicos e seus protocolos (por exemplo a suíte de protocolos TCP/IP).
